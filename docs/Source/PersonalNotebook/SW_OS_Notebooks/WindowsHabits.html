Windows 10 Notebook¶

This is a notebook of my window 10 usage habit. It helps me with windows 10 installation, management, software preferences and so on. One of the purposes of this notebook is a quick reminder for me to setup a Windows 10 computer after a clean installation.

Since Windows released Windows 10, it easily became my favorite operating system. I had zero frustration fighting with the driver, antivirus softwares and a bunch of other miscellaneous compatibility issues on Windows 10 OS. Microsoft Prevails!

Installation of Windows 10 (a few Notes)¶

  • Bootable USB Drive Creation: Windows 10 Installation media creation tool, Rufus

  • Use Rufus: Don’t use the media creation tool to create boot USB drive directly. Somehow, this tool won’t be able to change the usb drive to GPT. So the system installed in this way will mostly be Legacy mode with MBR partition. In order to install the windows 10 in GPT partition method using UEFI. You will have to create the installation media using Rufus. First download latest windows 10 image with Windows 10 Installation media creation tool.Then create bootable usb device with Rufus. Select GPT and UEFI during the creation. Don’t forget to change the BIOS to start from UEFI.

  • Be careful of the Sys Partition: When installing, take a second look at the partition table. Once the installation program install my system on two different hard-drives.EFI and System Recovery on one drive; MBR reserved and system main partition on another hard-drive. This will cause problem. If this happen, simply reinstall the system. Or, unplug all the other hard-drive in the system.

  • About Dual Boot: If dual-boot with ubuntu is required, try to install two system on different hard-drives. This will avoid lots of trouble when reinstall any one of them. The system will still be bootable with any hard-drive. If installing on the same hard-drive is necessary, then install the windows first,install the ubuntu manually with partitions created manually. After installation, the system will still boot from Windows. Ubuntu will not be seen when the system boots up. You will have to use easyBCD software to add entry of the ubuntu manually in windows. The benefit of this method is that you can always remove the ubuntu as you want. If you simply choose to install ubuntu on the side of windows as it suggested, it will be extremely painful to remove ubuntu only without damaging the boot of the windows. This is because GRUB will take control of the boot from windows.

  • GPT with UEFI: Always install your system with GPT partitioned hard-drive using UEFI booting. It is advanced method.

Attention

Delete OneDrive if you want to use GoogleDrive

Delete OneDrive after install the system. It came with Windows by default. I like it. However, during the daily usage, it kept fighting with my GoogleDrive. When both of them are on, my GoogleDrive icons notification will be disabled. Since GoogleDrive is my main cloud drive. I have to delete OneDrive.

Attention

Windows 10 & Ubuntu dual-boot Time Mismatch

In both Windows 10 and Ubuntu, they maintain the system bios RTC in the operating system. In windows 10, you can go to the time setting to Adjust date/time and turn on/off Set time automatically and Set time zone automatically. Windows by default will maintain the RTC clock on BIOS as the local time. Ubuntu on the other hand, will change the system timezone and local time on every bootup. Ubuntu maintain the system RTC time as UTC time and add a shift to this time according to the time zone you are in.

Due to different behaviours of these to operating systems, it will cause system time mismatch in Windows 10. A solution for solving this is to change the time setup in either Ubuntu or Windows. I prefer to change the time setup in Ubuntu since it is much easier and might cause less problem in the future. Simply type the following command in bash:

timedatectl set-local-rtc 1 --adjust-system-clock

For details, this site gives more detail about the time mismatch and its solutions.

Some Useful Tools on Windows¶

  • SpaceSniffer, with SpaceSniffer, you can easily view the space take on your disks. It is even better than the disk analysis tool default on the Ubuntu.

  • MobaXterm is a enhanced terminal for Windows with X11 server, tabbed SSH client, network tool and etc. My favorite SSH terminal compared to bitVise, TeraTerm, Putty.

  • FormatFactory is a tool to change the format of the files. It is no longer a clean install, you have to be careful of the additional softwares in the installation packages.
